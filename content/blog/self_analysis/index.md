### youtube
受動的に未知のストーリーを聞けるから楽に面白い

-> youtubeを見るときは「なんのために見るか」をはっきり考えてから見る

-> 考えるとゲーム配信やVtuberは基本的に生産性がない

-> ゲームを上手くなると言う理由で見るならセーフ,Vtuberはラジオとして聞けばセーフ?つまりなにか作業しながら音だけを聞く

-> (解決策) 「youtubeを見よう」と思った時「それは無駄なことだ」と即座に思う

-> 「「youtube」ではなく「プレイ解説」を見よう」はあり

-> Vtuberにおいては

| 作業 | 聞けるもの |
|:---|:---|
| 集中が必要 | なし |
| 集中しなくてもいい | 音楽 |
| 考えなくてもいい | Vtuber |

-> 考えなくてもいい作業なんてあるだろうか

-> Vtuberは見ない方がいい（もちろん新しい知見はあるが時間が溶けるデメリットが大きい）
